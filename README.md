# 辅助面板（Auxiliaryfunction）功能概览

## 面板目录
- 默认面板：物流站与采矿的参数配置与批量操作
- 渲染屏蔽：屏蔽模型、UI 和声音，支持一键全局屏蔽
- 戴森球蓝图：蓝图管理与应用，按层导入/删除，自动生成壳层
- 易用工具：蓝图优化、物流站信息显示、时间流速设置等
- 科技面板：科技队列管理、材料限制筛选、快速买断
- 战斗与导航：黑雾巢穴与通讯器的导航工具
- 蓝图批量配置：蓝图模式下的批量选择与删除、快捷键设置

## 默认面板
- 自动配置新运输站的开关与参数面板
- 参数调节（滑条）：配送机、运输船、最大充电功率、最远路程、曲速启用路程、起送量、翘曲填充数量、采矿速率
- 批量操作：一键填充配送机/运输船/翘曲器、批量配置物流站与采矿速率、铺满轨道采集器
- 自动科技队列：支持等级上限与沙盒秒解锁（开启后生效）

## 渲染屏蔽
- 模型屏蔽：戴森壳/云、研究站、传送带货物、黑雾、运输船与飞机、实体
- UI 屏蔽：电网覆盖、建筑栏提示、教学提示、顾问、里程碑、科技提示
- 声音屏蔽：关闭玩家走路/飞行声音
- 全局屏蔽：一键影响宇宙模拟器相关对象

## 戴森球蓝图
- 管理：打开蓝图文件夹、刷新列表、按类型筛选与选择
- 交互：复制到剪贴板、清空剪贴板、应用蓝图到当前选中戴森球
- 工具：自动生成最大半径壳层（最多 10 层）、删除全部空壳
- 层级操作：按层导入或删除

## 易用工具
- 蓝图优化：蓝图全选、蓝图删除、记住上次蓝图浏览路径
- 物流站信息：显示/隐藏，详细/简易模式切换
- 物流站复制：复制粘贴物品设置
- 快捷键设置：窗口快捷键捕获与确认
- 时间流速：倍率输入与快捷调整（范围 0.01–10）

## 科技面板
- “准备研究”列表：材料限制筛选，支持网格布局
- “科技”列表与“升级”列表：点击加入准备研究
- 图标栏：显示材料与解锁配方，支持买断科技

## 战斗与导航
- 黑雾巢穴自动导航：开关与跟随距离设置
- 火种列表：展示出发与目标星系、距离信息，支持导航
- 通讯器列表：展示与玩家距离，支持导航

## 蓝图批量配置
- 蓝图设置快捷键：切换修改与确认
- 批量选择：全选当前星球建筑并加入蓝图
- 批量删除：删除选中建筑并回收物资，重置蓝图预览

## 窗口操作与性能
- 拖动与缩放：仅在位置或尺寸变化时同步 UI，提高性能
- 尺寸限制：设置窗口最小宽高，避免过小或负值

## 快捷键与设置
- 主面板与蓝图面板快捷键互斥设置，支持数字、字母、功能键与修饰键
- 可选启用时间流速的快捷调节
